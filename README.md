# SmartSchoolDashbord 📚🤖

Plateforme de gestion scolaire intelligente pour les écoles du Bénin.

## 🚀 Description

SmartSchoolDashbooard est une plateforme SaaS qui permet aux écoles de :

* Gérer les élèves, classes et matières
* Suivre les notes et absences
* Générer automatiquement les bulletins PDF
* Simplifier la gestion administrative
* Préparer l'intégration future de l'IA éducative

---

## 🧱 Architecture

```
smartschool-ai/
├── frontend/ (Next.js)
└── backend/ (NestJS)
```

---

## ⚙️ Stack technique

* Frontend : Next.js + TypeScript + TailwindCSS
* Backend : NestJS + TypeScript
* Database : PostgreSQL
* Auth : JWT

---

## 📦 Installation

### 1. Cloner le projet

```bash
git clone <repo-url>
cd smartschooldashboard
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

### 3. Backend

```bash
cd backend
npm install
npm run start:dev
```
